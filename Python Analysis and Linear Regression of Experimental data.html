We verify orthogonality, train a linear regression model, and evaluate its ability to recover the true population coefficients.

3. Train/Test Split¶

A standard random split would destroy the balanced factorial structure and break orthogonality. Instead we split within each cell of the design — every unique combination of factors contributes exactly 80% of its rows to training and 20% to test, preserving balance in both sets.

In [22]:
from sklearn.model_selection import GroupShuffleSplit

# Create a cell identifier — every unique combination of the 4 factors
data['cell'] = (data['size_cat'].astype(str) + '_' + 
                data['dist_cat'].astype(str) + '_' + 
                data['renovated'].astype(str) + '_' + 
                data['location'].astype(str))

# Split within each cell (80/20) preserving balance
train_idx = []
test_idx  = []

for cell, group in data.groupby('cell'):
    n         = len(group)
    n_train   = int(n * 0.8)
    idx       = group.index.tolist()
    train_idx.extend(idx[:n_train])
    test_idx.extend(idx[n_train:])

train = data.loc[train_idx].reset_index(drop=True)
test  = data.loc[test_idx].reset_index(drop=True)

# Verify balance
print("Train size:", len(train))
print("Test size: ", len(test))
print("\nTrain cell counts:\n", train.groupby('cell').size().unique())
print("Test cell counts:\n",  test.groupby('cell').size().unique())

4. Model Training¶

We fit an Ordinary Least Squares linear regression model including the size:location interaction term, mirroring the true data generating process used in R. Because predictors are orthogonal, each coefficient is estimated independently without interference from the others.
